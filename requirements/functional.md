## Task 1.1 Functional Requirements (User Stories)

1. Browse meals - As a user, I want to browse meals offered by the catering service so that I can decide what I want to cook/order for.

2. View ingredients for a meal - As a user, I want to view the list of ingredients required for a selected meal so that I can see what is needed before ordering.

3. Order ingredients for a meal - As a user, I want to order ingredients for a selected meal so that I can get everything delivered without manually collecting items.

4. Schedule delivery - As a user, I want to schedule ingredient deliveries for same-day or next-day so that I can receive ingredients when I need them.

5. Track order status - As a user, I want to track the status of my orders so that I can know whether my ingredients are being processed or delivered.
